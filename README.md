### RASCUNHO:

- Projeto Open-source com foco em ser um repositório em ajudar pessoas com deficiência visual a aprenderem Lógica de Programação com Python durante a formação de Operador de Computador do programa Trilha TEC, projeto FLorescendo Talentos.

- Mensagem de um dos tutores do curso em relação á minha pergunta sobre o problema:
"A principal queixa envolto das dificuldades delas, era de não conseguir ver a transmissão de broadcasting, onde os materiais do curso eram passados, direito e não conseguir acompanhar a explicação corretamente"
    - Problemas identificados
        - Baixa percepção do conteúdo visual em tempo real: o material exibido na transmissão não tem escala/contraste suficientes para baixa visão.
        - Perda de acompanhamento da explicação: a estudante não consegue correlacionar, no mesmo ritmo, o que é mostrado com o que está sendo falado.
        - Dependência de mediação adicional para compreensão: sem adaptação, a aula fica inacessível e exige apoio externo constante para entender a atividade.
